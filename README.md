## DESCRIPTION

I successfully completed a **Text Summarization Tool** using **Natural Language Processing (NLP)** techniques as part of my AI Internship at CODTECH IT SOLUTIONS.

The main objective of this task was to develop a Python application capable of generating concise summaries from lengthy text while preserving the most important information. This project helped me understand the fundamentals of text processing, extractive summarization, and the practical application of NLP techniques.

For this project, I used **Python** as the programming language and **Visual Studio Code (VS Code)** as the development environment. VS Code provided a simple and efficient platform for writing, executing, and debugging the Python code.

The primary library used in this project was **Sumy**. I implemented the **LSA (Latent Semantic Analysis) Summarizer**, which identifies the most important sentences in a document based on semantic relationships.

The application accepts a paragraph as input, processes the text, and generates a concise summary. Finally, it displays both the original text and the summarized text in the terminal.

This project improved my understanding of Natural Language Processing, Python programming, and AI-based text analysis. It also enhanced my confidence in building practical AI applications using NLP libraries.
